# springboot-websocket-chat


# 功能列表

1、Web端的IM应用
2、支持上线、下线、实时在线提醒
3、单聊、群聊的建立
4、普通文字、表情、图片的传输（子定义富文本）
5、单人的顶级提醒，多对话的窗口的提醒
6、调用图灵机器人的自动回复演示

# 核心技术列表

1、websocket、sockjs、stomp
2、前端展示涉及的jquery、vue、elementUI、jquerybase64js
3、后端springboot、jsoup、spring-security、spring-websocket
